## Приложение для конвертации валют (usd, eur)

## Стэк технологий
При разработке данного приложения использовался следующий стэк технологий:

Python, FastAPI, SQLAlchemy, PostgreSQL, Alembic, Pydantic, Aiohttp, Asyncio, Unittest и др.

## Описание работы над проектом
- Разрабатывал код на фреймворке FastAPI
- Писал SQL запросы к PostgreSQL с использованием SQLAlchemy ORM
- Использовал Alembic для управления миграциями базы данных
- Применял Pydantic для валидации и сериализации данных
- Настраивал асинхронность (asyncio) для улучшения производительности работы кода
- Писал юниттесты для проверки корректности работы эндпоинтов
___

## Запуск приложения

Для запуска приложения нужно сделать следующее:

1. склонировать репозиторий:
    ```bash
    git clone https://github.com/Just-arman/currency_converter_app.git
    cd currency_converter_app
    ```

2. установить зависимости:
    ```bash
    pip install -r requirements.txt
    ```

3. запустить приложение:
    ```bash
    uvicorn app.main:app --reload
    ```
    Данную команду нужно запускать находясь в корневой директории проекта.
